## Design Foundation

**Visual Theme**
- **Primary Colors**: Deep navy (#1a2744) with gold accents (#d4af37)
- **Font**: Black Ops One for headings, clean sans-serif for body text
- **Dark/Light Mode**: Toggle in top-right corner with moon/sun icons
- **Animations**: Subtle fade-ins and slide transitions between sections

**Scrolling Experience**
- Vertical snap scrolling - each section fills the screen
- Smooth transitions when scrolling between sections
- Navigation dots on the side showing current section
